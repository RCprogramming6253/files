Learn the Basics of HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. Here are the essentials developers need to remember:

1. HTML Document Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    1.Always close the tags
    2. Never forget to save it
    3. Make sure to double-check if it is important!
  </body>
</html>
Tip: Every HTML file should start with <!DOCTYPE html> to declare the HTML version.

2. Common HTML Elements

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a href="url"> - Link
  • <img src="img.jpg" alt="desc"> - Image
  • <ul>, <ol>, <li> - Lists
  • <strong>, <em> - Bold and Italics
  • <br> - Line Break

3. Example: A Simple HTML Page

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is my first HTML page.</p>
    <a href="https://developer.mozilla.org/en-US/docs/Web/HTML">Learn more about HTML</a>
  </body>
</html>
